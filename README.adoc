== 환경설정

이 책은 ``asciidoc``형식의 텍스트 파일로 만들어졌고, ``asciidoctor``를 사용해서 책을 만든다. ``Asciidoctor``를 사용하려면 ruby를 설치해야한다.



=== ruby 설치하기

2가지만 기억하자.

1. Devkit이 포함된 ruby 설치하기
2. 마지막 검은색 명령창에서 ``1 + 엔터``, ``2 + 엔터``, ``3 + 엔터``


link:https://rubyinstaller.org/downloads/[루비 공식 홈페이지](link:https://rubyinstaller.org/downloads/[])에서  ``rubyinstaller-devkit-3.0.2-1-x64.exe``를 다운로드해서 설치한다.

마지막 검은색 명령창이 떴을 때가 가장 중요하다. 반드시 ``1 엔터``, ``2 엔터``, ``3 엔터`` 를 순서대로 입력해서 모두 설치해야 한다.

=== git 설치하기

이미 git이 설치되어 있겠지만.

git을 설치하는 이유는 git-bash를 사용하기 때문이다.

link:https://git-scm.com/[Git 공식 홈페이지(https://git-scm.com/)]에서 다운로드해서 설치하자.



=== make 설치하기

make도 설치해야한다. 이건 윈도우 패키지 관리자 Chocolatey를 사용해서 설치하는 것이 쉽다. choco를 설치하는 방법은 link:https://chocolatey.org/install[Chocolatey 공식 홈페이지(https://chocolatey.org/install)]를 참고하자.

[source,powershell]
----
choco install make
----



=== asciidoctor관련 패키지 설치

문서를 만들 컴파일 환경을 설정해야 하는데, progit2에서 봤던 방법대로는 왜인지 bundle에서 종속성 오류가 계속 발생하고 있다.
그냥 ``gem install 패키지명``명령으로 설치하면 오류가 생기지 않는다. 그래서 그냥 ``make bundle``명령으로 ``gem install``하도록 설정했다.

[source,bash]
----
gem install bundler
make bundle
make font
----

== 책 생성하기

모든 형식의 책을 생성하려면 ``make``만 입력한다.

[source,bash]
----
make
----

특정 형식의 책을 만들수도 있고:

[source,bash]
----
make html
make pdf
----

다른 것도 할 수 있다:

[source,bash]
----
make readme
make test
----

== 저장소를 처음 상태로 정리하기

저장소를 처음 상태로 정리하는 명령이다.

[source,bash]
----
make clean
----
